Modal windows are the new pop-up ad for today’s users. They often find these elements irritating because if not designed properly, it can disrupt user flow. However, using this concept to your advantage can help grab the user’s attention to direct them to important information.

Here are some scenarios that are appropriate for modal windows to be used in:

  • User input is needed
  • Showing additional information (within context and without)
  • Direct the user’s attention to something else

In order for a modal window to be well-designed, there are a few elements that must be included:

  • Escape button - give the user the option to close the modal
  • Descriptive title - gives context to why the modal appeared
  • Understandable button titles - descriptions of buttons should make sense in the context they are in
  • Sizing & location - content should fit in the modal, and it should not take up the entire screen
  • Lightbox effect - highlights the modal to draw attention to it
  • User initiated - let the user’s action guide them to the modal instead of surprising them with it

It is also important to note that modals do not work well with mobile because there is so little screen space. It is best to avoid using modals on mobile, and to find other alternatives to present the information instead. Also make sure to make the modals accessible by adding keyboard accessibility.
